# Meditrack 🏥💊

**Meditrack** is a full-stack web app to track medicines, monitor health metrics, and provide **24×7 healthcare services** like:

- First Aid 🩹  
- Doctor Support 👨‍⚕️  
- Elderly Care 👵  
- Home Nursing 🏠



- Medicine tracking 💊  
- Health metrics 📊  
- Doctor chat 💬  
- Home nursing 🏠
  

> ⚠️ Project is currently under development! Expect new features soon 🚀

---

## Features (Work in Progress)

- [x] Add & update medicine schedules  
- [x] Health metrics tracking (BP, sugar, heart rate)  
- [ ] Real-time chat with doctors  
- [ ] Push notifications & email alerts  
- [ ] AI-based health suggestions  
- [ ] Appointment booking system  

---

## Tech Stack

- **Frontend:** React / Next.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB / PostgreSQL  
- **Authentication:** JWT / OAuth2  
- **Deployment:** Vercel (Frontend), Heroku / Render (Backend)  

---

## Getting Started (Quick Setup)

```bash
git clone https://github.com/piyush200516/meditrack.git
cd meditrack

# Frontend
cd frontend
npm install
npm run dev

# Backend
cd ../backend
npm install
npm run dev

Open http://localhost:3000 to see your app in action.
